# Public Claim Ceiling

The public claim ceiling for `admissibility-kernel v0.1.0` is:

> This repository provides a public learning and inspection surface for context-to-consequence pre-commitment admissibility governance.

It may claim that it demonstrates:

- four governed outcomes: `EXECUTE`, `TRANSFORM`, `DEFER`, `REJECT`;
- three-state separation: current, projected, observed;
- public proxy observables: Ωᵣ, Θ, B_iso, residue;
- decision record production;
- local admissibility, recursive viability, and transformability preservation as separable concerns;
- a pedagogical bridge from human-before-the-loop judgement to governed execution.

It must not claim that it provides:

- production certification;
- legal or regulatory conclusions;
- deployment approval;
- safety guarantees;
- model-risk rating;
- organisational readiness scoring;
- Neverthought’s full internal kernel;
- calibrated thresholds or commercial adapters.

The public phrase to defend is:

```text
Context → Consequence → Admissibility → Commitment
```
